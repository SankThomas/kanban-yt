# Kanban Project

![kanban-demo](./kanban-demo.png)

Demo the app here: [https://tsbsankara-kanban-board.vercel.app/](https://tsbsankara-kanban-board.vercel.app/)

## Features

1. Drag and drop using Dnd-Kit
2. Create and delete boards
3. Create, reorder and delete tasks
4. Create, reorder and delete subtasks
5. Auth using Clerk
6. Convex reactive database
